# 会理市AI数字人导游系统 - 安装部署指南

## 一、系统要求

### 1.1 硬件要求
- CPU：2核以上（推荐4核）
- 内存：4GB以上（推荐8GB，需加载嵌入模型）
- 存储：10GB可用空间
- 网络：稳定的互联网连接

### 1.2 软件要求
- 操作系统：Windows 10/11、macOS 10.15+、Ubuntu 18.04+
- Python：3.8 或更高版本
- 浏览器：Chrome 90+（推荐）

### 1.3 外部服务
- 小米大模型API Key（必填，用于AI对话）
- 高德地图JS API Key（必填，用于地图显示和导航）

---

## 二、安装步骤

### 2.1 获取项目代码
将项目文件夹放置到目标目录，例如 `C:\huili_smart_guide`

### 2.2 创建Python虚拟环境
```bash
# 进入项目目录
cd C:\huili_smart_guide

# 创建虚拟环境
python -m venv .venv

# 激活虚拟环境
# Windows:
.venv\Scripts\activate
# Linux/macOS:
source .venv/bin/activate
```

### 2.3 安装依赖
```bash
pip install -r backend/requirements.txt
```

如果下载缓慢，可使用国内镜像：
```bash
pip install -r backend/requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 2.4 配置环境变量
在项目根目录创建 `.env` 文件，填入以下内容：

```ini
# 小米大模型API密钥（必填）
XIAOMI_API_KEY=你的小米API密钥

# 管理员密码（必填）
ADMIN_PASSWORD=你的管理员密码

# 服务器配置（可选，默认如下）
HOST=0.0.0.0
PORT=5000
```

### 2.5 配置高德地图Key
编辑 `frontend/index.html`，找到高德地图引用行，替换为你的Key：
```html
<script src="https://webapi.amap.com/maps?v=2.0&key=你的高德地图Key"></script>
```

### 2.6 构建知识库
```bash
# 完整构建
python knowledge_base/build_kb.py

# 或简化版构建（快速测试）
python knowledge_base/build_kb_simple.py
```

## 三、启动服务

### 3.1 启动后端
```bash
cd backend
python app.py
```

启动成功后会显示：
```
 * Running on http://127.0.0.1:5000
```

### 3.2 访问前端
后端启动后，直接在浏览器访问：
- 游客端：http://127.0.0.1:5000/static/index.html
- 管理端：http://127.0.0.1:5000/static/admin.html

或使用前端静态服务器：
```bash
cd frontend
python -m http.server 3000
```
- 游客端：http://localhost:3000
- 管理端：http://localhost:3000/admin.html

## 四、生产环境部署

### 4.1 使用Gunicorn（Linux/macOS）
```bash
pip install gunicorn

gunicorn "backend.app:app" \
    --workers=4 \
    --bind=0.0.0.0:5000 \
    --timeout=120
```

### 4.2 使用Nginx反向代理
```nginx
server {
    listen 80;
    server_name your-domain.com;

    location / {
        proxy_pass http://127.0.0.1:5000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }

    location /static {
        alias /path/to/frontend;
    }
}
```

### 4.3 Docker部署
```bash
# 构建镜像
docker build -t huili-smart-guide .

# 运行容器
docker run -d -p 5000:5000 \
  -e XIAOMI_API_KEY=你的密钥 \
  -e ADMIN_PASSWORD=你的密码 \
  huili-smart-guide
```

## 五、常见问题

### 5.1 端口被占用
```bash
# Windows: 查找并终止占用5000端口的进程
netstat -ano | findstr :5000
taskkill /PID 进程号 /F

# Linux/macOS
lsof -i :5000
kill -9 进程号
```

### 5.2 依赖安装失败
```bash
# 升级pip
pip install --upgrade pip

# 使用国内镜像
pip install -r backend/requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 5.3 小米API调用失败
- 检查 `.env` 中 `XIAOMI_API_KEY` 是否正确
- 检查网络连接
- 确认API账户余额充足

### 5.4 地图不显示
- 检查 `index.html` 中高德地图Key是否已替换
- 确认Key有效且未过期
- 检查浏览器控制台是否有跨域错误

### 5.5 知识库构建失败
- 确认 `knowledge_base/raw_data/` 目录中有文档
- 检查Python版本 >= 3.8
- 确认 `chromadb` 和 `sentence-transformers` 已安装
