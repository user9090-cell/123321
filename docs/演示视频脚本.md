# 会理市AI数字人导游系统 — 演示视频录制指南

## 录制工具推荐
- **Windows**: OBS Studio（免费）/ Xbox Game Bar（Win+G）
- **录制时长建议**: 3-5分钟
- **分辨率**: 1920×1080

---

## 演示流程（按此顺序录制）

### 第一部分：系统概览（30秒）

1. 打开浏览器访问 `http://localhost:5000`
2. 展示登录页面（暖色调古城风格）
3. 简单介绍："这是会理市AI数字人导游系统，集成了大语言模型、RAG知识库、Live2D数字人和语音交互"

### 第二部分：注册与登录（30秒）

1. 点击"注册"标签
2. 输入手机号和密码
3. 点击"注册"
4. 登录进入主页
5. 展示主界面：左侧Live2D数字人、右侧聊天区

### 第三部分：智能问答（60秒）

1. 输入"会理古城有什么好玩的？"并发送
2. 展示AI回复（RAG知识库精准回答）
3. 点击推荐问题"会理美食推荐"快速追问
4. 展示多轮对话能力

### 第四部分：语音交互（40秒）

1. 点击麦克风按钮
2. 浏览器请求麦克风权限 → 允许
3. 说"介绍一下龙肘山"
4. 展示语音自动转文字 → AI回复 → TTS语音播报

### 第五部分：地图导航（40秒）

1. 点击"附近景点"按钮
2. 展示Leaflet地图（以会理市为中心）
3. 点击"使用我的位置"获取GPS坐标
4. 点击"搜索"
5. 展示地图上标记的附近景点 + 下方列表

### 第六部分：其他功能（30秒）

1. 点击主题切换按钮（亮/暗主题）
2. 展示Live2D数字人点击互动（点击角色触发动作）
3. 展示满意度调研功能

### 第七部分：结束语（10秒）

"感谢观看！会理市AI数字人导游系统，让每一位游客都能拥有专属的AI导游。"

---

## 录制注意事项

1. **提前启动后端**：`cd backend && python app.py`
2. **关闭无关窗口和通知**
3. **使用麦克风时说话清晰**
4. **操作不要过快**，让评委看清每一步
5. **建议提前试录一次**确认流程顺畅
