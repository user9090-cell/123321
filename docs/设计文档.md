# 会理市AI数字人导游系统 - 产品说明书

## 一、产品简介

### 1.1 产品名称
会理市AI数字人导游系统

### 1.2 产品概述
会理市AI数字人导游系统是一款面向旅游场景的智能导览平台，融合了人工智能对话、3D数字人交互、语音识别与合成、地图导航等多项技术，为游客提供全天候、个性化的导游服务，同时为运营管理者提供数据监控与知识库管理能力。

### 1.3 适用对象
- **游客**：到访会理市的旅游者，通过游客端获取景点介绍、美食推荐、路线导航等服务
- **管理员**：系统运营人员，通过管理后台监控系统运行、管理知识库和用户

### 1.4 运行环境
- 浏览器：Chrome 90+、Edge 90+、Firefox 88+、Safari 14+
- 网络：需要互联网连接（调用AI大模型和地图服务）
- 后端：Python 3.8+，Flask框架

## 二、功能说明

### 2.1 游客端功能

#### 2.1.1 用户注册与登录
- 游客需使用手机号注册账号并设置密码
- 注册时可上传个人头像
- 登录后即可使用全部功能
- 退出登录后账号自动注销，需重新注册

#### 2.1.2 智能对话
- 在输入框输入问题，点击发送或按回车键即可提问
- 系统基于RAG检索增强技术，结合知识库内容生成准确回答
- 支持多轮对话，系统可理解上下文
- 回答下方显示相关推荐问题，点击可快速追问

#### 2.1.3 语音交互
- 点击输入框左侧麦克风按钮开启语音输入
- 说话结束后系统自动识别并转为文字发送
- 系统回复时可点击播放按钮进行语音播报
- 支持调节语速和音量

#### 2.1.4 附近景点与地图导航
- 点击输入框左侧地图按钮打开"附近景点"面板
- 面板顶部显示高德地图，标记用户位置和周边景点
- 可手动输入经纬度或点击"使用我的位置"自动定位
- 设置搜索半径后点击搜索，查看范围内景点列表
- 地图上点击景点标记可跳转高德地图导航
- 景点列表中每个卡片有"导航"按钮，点击同样跳转导航

#### 2.1.5 个人头像
- 注册时可选择上传头像（支持PNG/JPG/GIF/WEBP格式）
- 登录后点击导航栏头像可更换头像
- 头像会同步显示在聊天消息和用户信息栏中

#### 2.1.6 其他功能
- **主题切换**：点击导航栏月亮/太阳图标切换深色/浅色主题
- **清空对话**：点击用户信息栏垃圾桶图标清空聊天记录
- **满意度调研**：系统会弹出满意度调研弹窗，可对服务评分并提建议
- **使用帮助**：点击帮助图标查看功能说明

### 2.2 管理端功能

#### 2.2.1 登录
- 访问 admin.html 进入管理后台
- 输入管理员密码登录

#### 2.2.2 数据概览
- 仪表盘显示：总请求数、活跃会话数、平均响应时间、满意度
- 请求趋势折线图
- 用户反馈分布饼图

#### 2.2.3 实时状态监控
- API运行状态
- 数据库连接状态
- 缓存使用情况
- 应用版本信息

#### 2.2.4 知识库管理
- 查看知识库文档数量和状态
- 上传新文档（支持TXT/PDF/MD格式）
- 一键重建知识库

#### 2.2.5 用户管理
- 查看注册用户总数、已使用/未使用统计
- 用户列表显示手机号、注册时间、最后登录时间、头像
- 查看用户头像信息

#### 2.2.6 数据分析
- 高频问题排行
- 最近用户反馈记录

#### 2.2.7 系统设置
- AI模型参数配置（温度、最大令牌数等）
- 通知设置
- 安全设置（管理员密码修改）

## 四、操作流程

### 4.1 游客使用流程

```
注册账号 → 上传头像 → 登录 → 提问/语音输入 → 查看回答
                                    ↓
                        点击附近景点 → 查看地图 → 导航前往
                                    ↓
                        退出登录 → 账号自动注销
```

### 4.2 管理员操作流程

```
登录管理后台 → 查看数据概览 → 检查系统状态
                    ↓
        上传知识库文档 → 重建知识库
                    ↓
        查看用户列表 → 查看反馈 → 调整系统设置
```

## 五、技术架构

### 5.1 技术栈
| 层级 | 技术 |
|------|------|
| 前端 | HTML5 + CSS3 + JavaScript + 高德地图JS API |
| 后端 | Python + Flask |
| AI | 小米大模型 + RAG检索增强生成 |
| 向量数据库 | ChromaDB + BAAI/bge-small-zh-v1.5嵌入模型 |
| 缓存 | Redis / Fakeredis |
| 语音 | Web Speech API |

### 5.2 核心API

| 接口 | 方法 | 说明 |
|------|------|------|
| /api/register | POST | 用户注册 |
| /api/login | POST | 用户登录 |
| /api/logout | POST | 用户退出（注销账号） |
| /api/chat | POST | 智能对话 |
| /api/voice | POST | 语音输入 |
| /api/geo_nearby | GET | 附近景点查询 |
| /api/avatar/upload | POST | 头像上传 |
| /api/avatar/\<phone\> | GET | 获取头像 |
| /api/feedback | POST | 提交反馈 |
| /api/health | GET | 健康检查 |
| /api/user/stats | GET | 用户统计 |
| /api/stats | GET | 系统统计 |
| /api/admin/upload | POST | 知识库文档上传 |
| /api/admin/rebuild | POST | 重建知识库 |

## 六、注意事项

1. 退出登录后账号将被注销，无法恢复，需重新注册
2. 语音功能需要浏览器授权麦克风权限
3. 地图定位功能需要浏览器授权位置权限
4. 高德地图服务需要有效的API Key
5. AI对话依赖小米大模型API，需确保API Key有效且有余额
6. 知识库文档更新后需点击"重建知识库"才能生效
7. 建议使用Chrome浏览器以获得最佳体验
